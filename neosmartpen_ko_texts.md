본 마크는 손필기를 디지털로 저장해 주는 Ncode 적용 인증 마크입니다. 본 마크가 적용된 제품을 네오스마트펜과 함께 사용하면 손필기의 익숙함과 디지털의 편리함을 동시에 경험할 수 있습니다.

### 노트북 라인업

**N 메모노트 (단면만 인식)**
N 메모노트는 간단한 필기를 하기 위해 휴대가 용이한 작은 사이즈로 디자인 되었습니다. 순간 순간 떠오르는 아이디어를 기록하세요.
제품 정보: 유선 / 148 x 83mm / 50p / 미색 모조지 70g / 뜯어쓰는 노트

**N 블랭크 플래너 (만년달력)**
N 블랭크 플래너는 다양한 속지 구성으로 효율적인 스케쥴 관리가 가능합니다. 내가 원하는 월을 시작으로 계획을 세워보세요.
제품 정보: 150 x 210mm / 152p

**N 몰스킨 노트**
N 몰스킨 노트는 클래식 룰드 라지 사이즈로서, Ncode(TM)가 입혀진 미색 중성지와 발수처리된 블랙 하드 커버로 이루어져 있으며, 뒷면에는 대형 포켓이 내장되어 있어 실용성과 고급스러움을 동시에 느낄 수 있습니다.
제품 정보: 130 x 210mm / 240p / 미색 중성지 / 하드커버

**N 아이디어패드 미니**
N 아이디어패드 미니는 휴대가 편리한 리갈 패드 타입의 노트입니다. 절취선에 따라 뜯어 쓰기 용이하기 때문에 망설임 없이 마음껏 필기할 수 있습니다.
제품 정보: 127 x 200 mm / 100p / 백상지 90g

**N 컬리지노트**
N 컬리지노트는 3개의 묶음 형태로 구성되어 있습니다. 선이 그어져있는 N 컬리지노트는 필기량이 많은 학생들에게 최적의 노트입니다.
제품 정보: 216 x 280 mm / 200p / 미색 모조지 80g / 3개 묶음

**N 포켓노트 (양면으로 인식)**
N 포켓노트는 어떤 주머니에도 쏙 들어가게끔 디자인 되었습니다. 옷 주머니나 N holder에 넣어 휴대하세요.
제품 정보: 격자무늬 / 144 x 83mm / 64p / 미색 모조지 80g / 중철 제본

**N 링노트**
N 링노트는 이동하며 많은 필기를 해야하는 사용자를 위해 탄생한 노트입니다. A5 크기의 가벼운 노트를 휴대하며 생각을 정리하세요.
제품 정보: 유선 / 210 x 150mm / 152p / 백색 모조지 80g / 링 제본

**N 프로페셔널**
N 프로페셔널은 고급스러운 재질로 디자인 되어, 다양한 용도에 맞게 제작 되었습니다. 유선과 무지로 제작된 노트가 제공되며, 명함을 꼽을 수 있는 포켓도 있습니다.
제품 정보: 140 x 205mm / 250 p / 미색 모조지 80g / 유선노트 & 무지종이

**N 아이디어패드**
N 아이디어패드는 리갈패드 타입의 유선 노트로 강의, 회의 등의 기록을 위한 최적의 노트입니다.
제품 정보: 210 x 297mm (A4) / 100p / 백상지 80g / 리갈 패드

**N 홀더**
N 홀더는 네오스마트펜n2를 안전하고 편리하게 보관할 수 있으며, 메모노트, 포켓노트는 물론이고 리필펜팁, 명함, 카드 등 필요한 물품을 휴대할 수 있도록 만들어졌습니다.
제품 정보: 메모노트 1권 / 115 x 170 mm / PU+SUEDE+PVC

## 14. DIY Ncode 노트북 (/ncode-pdf/)

**페이지 제목:** Ncode PDF | 네오스마트펜 공식 페이지
**URL:** https://neosmartpen.co.kr/ncode-pdf/

DIY NCODE
노트북

DIY 프린팅
원하는 포맷을 직접 다운로드 하여 프린트

넉넉한 페이지
각 노트들은 50페이지로 구성

다양한 종류
다양한 종류의 노트를 제공

NCODE 노트 용지
다운로드 (15종)

사용 시 주의사항
1. Ncode PDF는 컬러 레이저 프린터의 사용을 권장합니다. (잉크젯 프린터나 흑백 레이저 프린터 사용 시, 정교한 Ncode 인쇄 및 Ncode와 포맷 영역 간 오버랩이 발생하여 올바른 인식을 보증할 수 없습니다)

2. Ncode의 인쇄 품질을 위하여 PCL 혹은 PS 드라이버를 설치하여 인쇄하여 주시기 바랍니다.(PCL, PS 드라이버는 프린터기 제조사에 문의하여 주세요)

3. 흑백 레이저 프린터 사용 시, plain 용지 사용을 권장합니다.

이건 꼭 알아 두세요

시험 인쇄를 먼저!
Ncode 프린팅이 지원되지 않는 프린터가 있을 수 있습니다. 전체 페이지를 프린트 하시기 전에 일부 페이지를 프린트해서 인식이 잘 되는지 확인하신 후에 전체 페이지를 프린트해서 쓰는 것을 권장 합니다.

NEO STUDIO
스마트펜 입력 활성/비활성
네오스튜디오 앱에는 각 노트에 대해서 스마트펜 입력을 활성 혹은 비활성화 시키는 기능이 있습니다. 이 기능을 사용하여 출력한 노트의 페이지에 썼을 때 겹침을 방지할 수 있습니다.
